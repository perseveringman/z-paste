# v1.1.0 发布设计文档

日期：2026-02-28

## 背景

stash 当前版本为 1.0.1，README 为开发者技术架构文档（架构图、数据库 schema），不适合普通用户阅读。本次发布目标是：重写 README 为用户友好的产品展示页，升版本至 1.1.0，同步更新文档站并触发 GitHub Pages 部署。

## 工作项

### 1. 重写 README.md

面向普通 macOS 用户，结构：
- 产品标题 + 一句话介绍 + 徽章（版本/平台）
- 下载区域（区分 Apple Silicon / Intel，链接到 Releases）
- 功能特性（6 个核心功能，表情图标 + 一句话，去掉技术词汇）
- 快速上手（3 步：下载 → 拖入 Applications → 按 Shift+Cmd+V）
- 文档链接（GitHub Pages 文档站）
- 系统要求 + License

### 2. 版本号更新

- `package.json` version: `"1.0.1"` → `"1.1.0"`
- `website/docs/changelog.md` 新增 v1.1.0 条目

### 3. 发布 v1.1.0

- commit 所有变更
- `git tag v1.1.0`
- `git push origin main --tags`
- 触发 `release.yml` 自动构建 `.dmg` 并创建 GitHub Release

### 4. 文档站自动部署

- push main 触发 `docs.yml`，重新构建文档站（含更新后的 changelog）并推送到 `gh-pages`
