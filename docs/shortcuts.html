快捷键参考

全局快捷键

这些快捷键在任何应用程序中均可使用（需要「输入监控」权限）：

快捷键功能
Shift+Cmd+V唤起 / 隐藏剪切板面板
Shift+Cmd+P打开密码保险箱
Alt+W显示 / 隐藏小组件（Mini Widget）
Alt+1 ~ Alt+5快速粘贴小组件中的第 1-5 条目

:::tip 全局快捷键可在「设置 → 快捷键」中自定义。 :::

剪切板面板内快捷键

当剪切板面板处于焦点时有效：

快捷键功能
1 ~ 9快速粘贴对应位置的条目
↑ / ↓在列表中上下移动
Enter粘贴当前选中条目
Esc关闭面板
Cmd+F将光标移到搜索框
Delete删除当前选中条目

密码保险箱内快捷键

快捷键功能
Cmd+N新建密码条目
Cmd+C（在条目上）复制密码到剪切板
Cmd+B（在条目上）复制用户名到剪切板
Esc关闭保险箱 / 取消编辑
