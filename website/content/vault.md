# 密码保险箱

## 初始化保险箱

首次使用保险箱前需要设置主密码：

1. 按 **Shift+Cmd+P** 打开保险箱，或从菜单栏图标进入
2. 点击「设置主密码」
3. 输入一个强密码（建议 12 位以上，包含大小写字母、数字、符号）
4. 可选：设置密码提示问题（仅显示提示，不保存答案）
5. 妥善保存显示的「恢复密钥」，这是忘记主密码时唯一的找回方式

:::danger 重要
主密码无法找回。若遗失主密码且无恢复密钥，保险箱数据将永久无法访问。
:::

## 解锁保险箱

设置主密码后，每次进入保险箱时需要验证身份：

- **Touch ID**（若 Mac 支持并已启用）：按提示触摸指纹传感器
- **主密码**：手动输入主密码

在「设置 → 安全」中可以选择：
- **宽松模式**：Touch ID 失败后自动回退到密码输入
- **严格模式**：必须输入主密码，不接受 Touch ID

## 添加密码条目

1. 解锁保险箱后，点击右上角 **+** 按钮
2. 填写以下信息：
   - 标题（如「Gmail」「公司 VPN」）
   - 用户名 / 邮箱
   - 密码（可点击「生成密码」自动生成强密码）
   - 网址（可选）
   - 备注（可选）
3. 点击「保存」

## 密码生成器

在添加/编辑条目时，点击密码字段旁的「生成」按钮，可以：

- 设置密码长度（默认 16 位）
- 选择包含大写字母、小写字母、数字、特殊符号

## TOTP 两步验证码

若某账户启用了 TOTP 两步验证（如 Google Authenticator 等），可以在条目中添加 TOTP 密钥：

1. 编辑密码条目，找到「两步验证」字段
2. 扫描二维码或手动输入密钥（通常以 `base32` 格式提供）
3. 保存后，该条目将显示实时动态验证码（每 30 秒更新）

## 自动填写（Auto-Type）

Z-Paste 可以自动将用户名和密码填入当前激活窗口：

1. 在保险箱中找到对应条目
2. 点击「自动填写」或使用对应快捷键
3. Z-Paste 会自动切换到上一个活跃窗口并依次输入用户名、Tab、密码

:::note
自动填写需要授予「辅助功能」权限。
:::

## 自动锁定

保险箱在以下情况自动锁定：

- 屏幕保护程序启动或屏幕锁定
- 窗口失去焦点（严格模式下）
- 设置的超时时间到达（在「设置 → 安全」中配置，默认 5 分钟）

## 安全审计日志

在「设置 → 安全 → 审计日志」中可以查看所有保险箱访问记录（解锁时间、操作类型），帮助发现异常访问。
