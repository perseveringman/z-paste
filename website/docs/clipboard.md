# 剪切板管理

## 基本使用

Z-Paste 在后台自动监控剪切板，每次您按下 **Cmd+C** 复制内容时，Z-Paste 都会将其保存到历史记录中。

### 唤起面板

按 **Shift+Cmd+V** 唤起剪切板面板。

### 选择并粘贴

| 操作 | 说明 |
|---|---|
| 鼠标点击 | 选中该条目并粘贴到上一个活跃窗口 |
| 数字键 1-9 | 快速粘贴对应位置的条目 |
| 方向键 ↑↓ | 在列表中移动 |
| Enter | 粘贴当前选中条目 |
| Esc | 关闭面板 |

## 搜索

在面板顶部输入框中输入关键词即可进行模糊搜索，支持搜索内容文本。

## 内容类型

Z-Paste 会自动识别并标注以下内容类型：

| 类型 | 说明 |
|---|---|
| 文本 | 普通文字 |
| 链接 | 以 `http://` 或 `https://` 开头的 URL |
| 代码 | 代码片段（自动语法高亮） |
| JSON | JSON 数据（自动格式化预览） |
| 颜色 | HEX/RGB 颜色值（显示色块预览） |
| Base64 | Base64 编码字符串 |
| 图片 | 截图或复制的图片 |
| 文件路径 | 文件系统路径 |

## 收藏与置顶

- **收藏**：悬停在条目上，点击星形图标，收藏的条目不会被自动清除
- **置顶**：点击图钉图标，置顶的条目始终显示在列表顶部

## 序列粘贴

序列粘贴允许您将多个条目排队，按顺序依次粘贴：

1. 在剪切板面板中，依次选择多个条目并标记为序列
2. 每次按 **Shift+Cmd+V** 触发粘贴时，自动粘贴队列中的下一条

## 批量粘贴

选择多个条目后，可将它们合并为一段文字一次性粘贴。

## 模板片段

在「设置 → 模板」中可以保存常用文字模板（如邮件签名、代码片段），在面板中通过 `/` 触发快速插入。

## iCloud 同步

在「设置 → 通用」中开启 **iCloud Drive 同步**，剪切板历史将同步到您的所有 Mac 设备。

:::note
同步仅包含文本类型历史，图片不会同步。
:::

## 清除历史

在「设置 → 隐私」中可以手动清除全部历史记录，或设置自动清除策略（如保留最近 30 天）。
