# 常见问题

## 安装与启动

### Z-Paste 打不开，提示「无法验证开发者」

这是 macOS Gatekeeper 的安全机制。解决方法：

1. 打开 **系统设置 → 隐私与安全性**
2. 向下滚动找到「已阻止使用 Z-Paste，因为来自未知开发者」
3. 点击 **仍然打开**，再次确认

### Z-Paste 无法粘贴到其他应用

请检查「辅助功能」权限是否已授予：

1. **系统设置 → 隐私与安全性 → 辅助功能**
2. 确认列表中有 **Z-Paste** 并已打开开关
3. 若不在列表，点击左下角 **+** 号手动添加

### 快捷键 Shift+Cmd+V 没有反应

请检查「输入监控」权限：

1. **系统设置 → 隐私与安全性 → 输入监控**
2. 确认 **Z-Paste** 已被勾选
3. 权限更改后可能需要重启 Z-Paste

## 剪切板功能

### 为什么某些内容没有被记录？

Z-Paste 不记录以下内容（出于隐私和安全考虑）：

- 密码管理器（1Password、Bitwarden 等）复制的内容（这些应用会标记剪切板来源为敏感）
- 长度超过 1MB 的内容

### 如何防止某些应用的内容被记录？

在「设置 → 隐私 → 排除应用」中添加不想被监控的应用。

### 历史记录保存多久？

默认保存最近 **500 条**记录。可在「设置 → 通用」中调整保存数量或设置按时间清除。

## 密码保险箱

### 忘记了主密码怎么办？

在初始化保险箱时保存的「恢复密钥」是唯一找回途径。使用恢复密钥可以重置主密码。

如果同时丢失了主密码和恢复密钥，保险箱数据将**无法恢复**。

### 保险箱数据存在哪里？

存储在本机：`~/Library/Application Support/Z-Paste/vault.db`，使用 AES-256 加密，不会上传到任何服务器。

### 更换 Mac 后能迁移数据吗？

可以。将以下目录完整复制到新 Mac 同样路径即可：

```
~/Library/Application Support/Z-Paste/
```

## 其他

### Z-Paste 会影响系统性能吗？

Z-Paste 使用轮询方式监控剪切板（每 500ms 检查一次），CPU 占用极低，常驻内存约 50-80MB。

### 如何彻底卸载？

1. 将 `/Applications/Z-Paste.app` 移入废纸篓
2. 删除用户数据目录：`~/Library/Application Support/Z-Paste/`
3. 清空废纸篓
