# 快捷键参考

## 全局快捷键

这些快捷键在任何应用程序中均可使用（需要「输入监控」权限）：

| 快捷键 | 功能 |
|---|---|
| **Shift+Cmd+V** | 唤起 / 隐藏剪切板面板 |
| **Shift+Cmd+P** | 打开密码保险箱 |
| **Alt+W** | 显示 / 隐藏小组件（Mini Widget） |
| **Alt+1 ~ Alt+5** | 快速粘贴小组件中的第 1-5 条目 |

:::tip
全局快捷键可在「设置 → 快捷键」中自定义。
:::

## 剪切板面板内快捷键

当剪切板面板处于焦点时有效：

| 快捷键 | 功能 |
|---|---|
| **1 ~ 9** | 快速粘贴对应位置的条目 |
| **↑ / ↓** | 在列表中上下移动 |
| **Enter** | 粘贴当前选中条目 |
| **Esc** | 关闭面板 |
| **Cmd+F** | 将光标移到搜索框 |
| **Delete** | 删除当前选中条目 |

## 密码保险箱内快捷键

| 快捷键 | 功能 |
|---|---|
| **Cmd+N** | 新建密码条目 |
| **Cmd+C**（在条目上） | 复制密码到剪切板 |
| **Cmd+B**（在条目上） | 复制用户名到剪切板 |
| **Esc** | 关闭保险箱 / 取消编辑 |
